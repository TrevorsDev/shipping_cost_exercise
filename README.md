# Sal's Shipping Cost Calculator

A Python program based on a Codecademy exercise, enhanced to create a robust, interactive tool for calculating shipping costs using different shipping methods.

## Features

- Calculates shipping costs for:
  - Ground Shipping
  - Ground Shipping Premium
  - Drone Shipping
- User input is normalized for consistency (case and spaces)
- Clear user feedback for invalid input
- Modular code structure using functions

## How to Run

1. Clone this repository or download the code.
2. Open a terminal and navigate to the project folder.
3. Run the program with:
   ```bash
   python3 shipping_cost_exercise.py
